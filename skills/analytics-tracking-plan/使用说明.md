# 埋点 Skill 使用说明

## 快速开始

1. 复制本文件夹到 `<项目>/.cursor/skills/analytics-tracking-plan/`
2. Cursor 对话：`@analytics-tracking-plan 帮我出埋点` + 页面截图
3. 打开生成的 `{page_id}_埋点开发计划.html`

## 粘贴到 Excel

1. 点击右上角 **「一键复制数据行」**
2. 在 Excel 已有表头的「开发计划」Sheet 中 Ctrl+V
3. 复制内容**不含表头**，仅数据行（7 列）

## 示例文件

| 文件 | 说明 |
|------|------|
| `examples/discover_page_埋点开发计划.html` | 发现页，14 条 |
| `examples/work_detail_page_埋点开发计划.html` | 作品详情页，11 条，全事件带 work_id |

## 维护命令

```bash
# 在 skill 目录下执行
python scripts/update_reference.py    # Excel 更新后
python scripts/sync_html_shell.py     # 模板外壳更新后
```

详细说明见 [README.md](README.md)。
